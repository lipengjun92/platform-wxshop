- **分层架构**：采用标准的 Controller-Service-Dao 三层结构，基于 Spring Boot 构建。
- **安全认证**：集成 Apache Shiro 实现基于 OAuth2 Token 的无状态认证与权限控制（`ShiroConfig`, `Oauth2Filter`）。
- **数据持久化**：使用 MyBatis-Plus 作为 ORM 框架，配合 Druid 动态数据源；通过 `DataScopeInnerInterceptor` 实现基于机构的数据权限自动过滤。
- **功能模块**：划分为 `sys`（系统基础）、`mall`（商城业务）、`job`（Quartz 定时任务）、`gen`（Velocity 代码生成）、`wx`（微信生态集成）及 `oss`（对象存储）等子域。
- **切面增强**：利用 AOP 实现操作日志记录（`SysLogAspect`）与请求 URI 拦截。

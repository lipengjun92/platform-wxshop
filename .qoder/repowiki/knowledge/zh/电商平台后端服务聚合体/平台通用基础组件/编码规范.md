- 统一使用 `RestResponse<T>` 作为 API 响应载体，通过静态工厂方法 `ok()` 和 `fail()` 构建结果，避免直接返回原始对象。
- 全局异常处理器 `BusinessExceptionHandler` 集中捕获各类异常（业务、参数、权限、微信等），并转换为标准的 `RestResponse` 格式返回。
- Web 安全过滤器（XSS、SQL注入）通过 `FilterRegistrationBean` 在 `FilterConfig` 中统一注册，并设置作用于所有请求路径 `/*`。
- 工具类（Utils）均采用静态方法设计，无状态且线程安全，如 `StringUtils`, `DateUtils`, `IpUtils` 等。

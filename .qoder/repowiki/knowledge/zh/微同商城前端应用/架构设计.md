- **框架与入口**：采用 Vue.js + uni-app 架构，以 `main.js` 为应用入口，`App.vue` 管理全局生命周期（如版本更新检测、错误监控）及全局样式。
- **路由与导航**：通过 `pages.json` 集中配置页面路由、底部 TabBar（首页、专题、分类、购物车、我的）及窗口样式，支持分包加载。
- **状态管理**：使用 Vuex (`store/index.js`) 维护全局状态，如网络连接状态及应用版本信息。
- **模块化设计**：`utils/api.js` 统一定义后端 API 接口路径；`components` 目录封装通用 UI 组件（如富文本解析 uParse）；`skills` 目录集成 AI 技能模块（导购、结算、订单），通过 MCP 协议与小程序环境交互。
- **多端适配**：`manifest.json` 配置了微信、支付宝、百度等多端平台的特定参数（如 AppID、权限、SDK 配置），实现一次开发多端运行。
